# Vue2-MV

> 基于Vue2实现的网易云音乐MV的webapp。[在线地址](https://mv.mrcxh.com/)

### 主页功能
- 浏览MV列表
- 搜索MV
- 观看MV
- 浏览MV评论

### 技术栈
vue2 + vue-router + webpack + ES6 + sass + rem + html5 video

## 安装运行（Build Setup）

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:5000
npm run dev

# build for production with minification
npm run build

```
